# Public assets

Drop your static assets here. They will be served from `/` at runtime.

## Required

- `resume.pdf` — your real resume. The `Resume` section embeds this file via
  `<object>` / `<iframe>` and exposes a download button. Replace the placeholder
  before deploying.

## Optional

- `favicon.ico` — browser tab icon. A lightweight SVG (`icon.svg`) is already
  provided; Next.js will use it automatically.
- `og-image.png` — 1200x630 social preview image.
