# Project cover images

Project cards in `data/projects.ts` reference images from this folder.

## Files

- `insurance-portal.svg` — tech-themed cover (blue/indigo shield + document cards).
- `inventory-management-system.svg` — tech-themed cover (warehouse boxes + scanner beam).

## Using real screenshots

If you'd rather show actual screenshots, drop them in here (e.g.
`insurance-portal.png`) at a **16:10 aspect ratio** (1600×1000 recommended)
and update the `image` field in `data/projects.ts` accordingly.
